## 🚀 Exercícios de Lógica de Programação e Algoritmos

Este repositório foi criado para documentar e organizar resoluções de problemas de lógica de programação, focando na transição entre o raciocínio visual (fluxogramas) e o desenvolvimento técnico (pseudocódigo).

## 🛠️ Tecnologias e Ferramentas
Para o desenvolvimento deste projeto, utilizei:

*   **Flowgorithm**: Ferramenta gráfica para criação de fluxogramas, permitindo a visualização clara da lógica de execução e controle de fluxo.
*   **VisualG**: Ambiente para edição e execução de algoritmos em pseudocódigo, essencial para o entendimento de estruturas de dados e controle.

## 📂 Organização do Repositório
O projeto está dividido em **7 pastas numeradas**, facilitando a navegação e o estudo de cada desafio proposto. Cada diretório contém:

*   **Questão X / Algoritmo**: Arquivo fonte `.alg` desenvolvido no VisualG.
*   **Questão X / Fluxograma**: Arquivo `.fprg` gerado no Flowgorithm.

## 💡 Objetivos
- Praticar a interpretação de enunciados e a criação de soluções lógicas.
- Consolidar conceitos de variáveis, operadores, estruturas condicionais e de repetição.
- Demonstrar a aplicação prática da lógica antes da codificação em linguagens de alto nível.
